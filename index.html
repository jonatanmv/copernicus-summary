Copernicus Project Summary and References

Since 24 January 2023 a new Copernicus Data Space Ecosystem has been launched to provide access to all Sentinel data with new features for visualisation and data processing.

Copernicus provide knowledge, but all start with DATA. The adopted policy under this project is based on a Full, Free and Open access to all that data.

Copernicus Satellite Data Sources

  • Managed by ESA (European Space Agency).
    • Open Access Hub (SCI Hub). The Copernicus Open Access Hub provides complete, free and open access to Sentinel data products....
      https://scihub.copernicus.eu
    • Copernicus Space Component Data Access (CSCDA). CSCDA service has Differentiated Access Levels.
      https://spacedata.copernicus.eu
  • Managed by EUMETSAT (European Organisation for the Exploitation of Meteorological Satellites).
    • EUMETCast. Service is mostly Free and Open.
      https://www.eumetsat.int/eumetcast
    • Copernicus Online Data Access (CODA). The Copernicus Online Data Access (CODA) service stop operating on 27 September 2022.
